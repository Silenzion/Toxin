#Учебный проект от FSD.
Респонсив вёрстка сатйи по поиску номеров в отеле toxin.

Распаковка проекта: 

**npm install**

Запуск режима разработки:
**npm run dev**

Сборка:
**npm run build**

Публикация на gh-pages:
**npm run deploy**
